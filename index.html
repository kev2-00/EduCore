Core modules

Five focused modules that cover the school's highest-friction workflows.

The MVP is intentionally narrow so each core workflow is easy to adopt, easy to train, and easy to maintain.

✓

Attendance

Teachers submit attendance once per day per class, with offline storage and background sync on reconnect.

₥

Fee Management

Record manual cash payments or reconcile M-Pesa transactions into one unified school fee ledger.

☏

Parent Communication

Send portal links, fee reminders, attendance alerts, and semester access notices through email and SMS fallback.

R

Report Cards & Results

Enter term results, generate report cards, and publish them to the parent portal in a mobile-friendly format.

⚑

Student Risk Dashboard

Use simple rule-based flags for absenteeism, fee arrears, and teacher concern notes.

CSV

Fast Onboarding

Bulk import students and parents using CSV so a new school can be set up in under one hour.

Interfaces

Three interfaces, each matched to the user's real device and workflow.

Instead of building a heavy app ecosystem, the platform uses one web-first architecture across teachers, admins, and parents.

T

Teacher PWA

Installable from Chrome on Android or Windows, optimized for end-of-day attendance and teacher notes.

D

Admin Dashboard

A full web dashboard for bursars, secretaries, and school leaders handling fees, risk flags, and reports.

P

Parent Portal

A simple mobile web portal with semester OTP access for balances, attendance summaries, and report cards.

Roadmap

Building in three focused phases.

Each phase adds capability without disrupting what's already working for schools in the field.

Phase 1 — MVP

Student and parent registration, offline attendance, fee ledger, sync dashboard, OTP portal, and pilot schools.

Phase 2 — Growth

Commercial launch, Swahili support, premium PDF exports, and multi-campus management for growing school networks.

Phase 3 — Scale

Regional expansion, localised payment rails, school health metrics, and possible integration with wider education systems.
